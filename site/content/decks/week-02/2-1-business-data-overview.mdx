<Slide title="Data Dimensions: The 5 Vs" icon="database" key="Modern business data is characterized by five distinct dimensions that drive complexity and value.">

<Bullets items={[
  { l: 1, h: "**Volume** — The sheer scale of data generated (gigabytes to petabytes)." },
  { l: 1, h: "**Velocity** — The speed at which new data is created and needs to be processed (e.g., real-time credit card checks)." },
  { l: 1, h: "**Variety** — The different forms data takes (tables, text, PDFs, images)." },
  { l: 1, h: "**Veracity** — The reliability, accuracy, and quality of the data (critical for audit trails)." },
  { l: 1, h: "**Value** — The ultimate business utility and insights derived from analyzing the data." }
]} />

<Slide title="Structured vs. Unstructured Data" icon="layers" key="Business data is split between structured tables and unstructured narrative documents.">

<Bullets items={[

  { l: 2, h: "Rigid, fixed-column records like General Ledgers and inventory listings." },
  { l: 2, h: "Stored in relational databases (SQL) and ERP/CRM/HRM platforms." },

<Slide title="Introduction to Storage Concepts" icon="database" key="The storage medium selected depends on the structure of the data and when it needs to be processed.">

<Bullets items={[
  { l: 1, h: "**Excel Spreadsheets** — Accessible flat files, convenient for small-scale analysis but highly fragile." },
  { l: 1, h: "**Relational Databases** — Organized tables optimized for high-speed queries and data integrity." },
  { l: 1, h: "**Data Warehouses** — Central hubs of structured databases utilizing *schema-on-write* controls." },
  { l: 1, h: "**Data Lakes** — Repositories of raw, multi-format datasets utilizing *schema-on-read* flexibility." }
]} />

</Slide>

<Slide title="Industrial Scale Issues" icon="layers" key="As data volume grows, firms face mounting governance, security, and cost pressures.">

<Bullets items={[

  { l: 1, h: "**Security & Privacy Compliance** — Protecting personal customer data and keeping strong audit logs." },
  { l: 1, h: "**Operational Cost Control** — Shifting hardware capital costs (CapEx) to monthly usage-based hosting costs (OpEx)." },
  { l: 1, h: "**Integration Imperative** — Automation and automated ETL (Extract, Transform, Load) are required to get a positive ROI." }
