<Bullets items={[

<Slide title="Financial & Audit Caveats" icon="layers" key="Cloud environments resolve physical resource limits but introduce billing volatility and right-to-audit issues.">

<Bullets items={[
  { l: 1, h: "**CapEx to OpEx Shift** — Upfront server equipment purchases (Capital Expense) are replaced by monthly utility bills (Operating Expense)." },
  { l: 2, h: "Can dramatically lower fixed assets on the balance sheet but increases monthly operating costs." },
  { l: 1, h: "**IT Cost Paradox** — 66% of executives report that migrating to the cloud did not lower IT costs (KPMG Survey)." },
  { l: 2, h: "Neglecting to shut down idle database instances run overnight can lead to massive cost overruns." },
  { l: 1, h: "**Audit Trail Concerns** — Relying on external cloud hosts can create 'right-to-audit' verification hurdles." }
]} />
