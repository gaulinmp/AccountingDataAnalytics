<Slide title="AICPA Standards Framework" icon="layers" key="The ADS framework covers a core base standard and five operational transaction subledgers.">

<Bullets items={[
  { l: 1, h: "**Base Standard** — Common master data including user tables, business units, segments, and tax tables." },
  { l: 1, h: "**General Ledger Standard** — Chart of accounts, journal entries, and historical trial balances." },
  { l: 1, h: "**Order to Cash (O2C) Subledger** — Customer sales, credit management, billing, and cash collections." },
  { l: 1, h: "**Procure to Pay (P2P) Subledger** — Purchasing requests, purchase orders, vendor invoices, and cash disbursements." },
  { l: 1, h: "**Inventory Subledger** — Inventory items, locations, physical counts, and cost updates." },
  { l: 1, h: "**Fixed Asset Subledger** — Asset registry, acquisitions, depreciation schedules, and disposals." }
]} />

</Slide>
