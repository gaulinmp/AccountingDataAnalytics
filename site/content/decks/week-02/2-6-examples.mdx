<Slide title="Standard Hands-On Workflows" icon="database" key="Typical business users extract raw tabular data to run manual manipulations in Excel.">

<Bullets items={[
  { l: 1, h: "**Requesting raw data** — Asking IT for a transactional database dump (CSV/Excel) covering a specific period." },

  { l: 1, h: "**Reporting** — Delivering the completed static spreadsheet to the executive team." }

<Bullets items={[

<Slide title="Key Takeaways" icon="layers" key="Analytics projects fail without a deep understanding of the business process that generated the data.">

<Bullets items={[
  { l: 1, h: "**Data is Messy** — Underestimating the time needed for cleaning and merging is the primary cause of project delays." },
  { l: 1, h: "**Data Generating Process (DGP)** — You must understand the actual business activities that create the records:" },
  { l: 2, h: "How are timestamps recorded? What triggers a record deletion? Who has authority to override values?" },
  { l: 1, h: "**Garbage In, Garbage Out** — High-end statistical models and LLMs are useless if the underlying data lacks integrity." }
]} />
