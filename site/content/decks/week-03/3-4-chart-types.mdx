<Slide title="Bar Charts">

- Used for quantitative or categorical data
- Facilitate comparison of data across categories
- Bars can be grouped for hierarchical / multi categories
- Stacked bar charts often used to demonstrate composition (e.g. % of total)
- Note: histograms sure look like bar charts, but some data analysts will argue they are not bar charts (see Leland Wilkinson, The Grammar of Graphics)

</Slide>

<Slide title="Histogram">

- Used to illustrate distributions
  - Very useful as a first check of data
- Non-parametric (no assumptions about distributions)
- Number of bins can be adjusted to average/smooth out data

</Slide>

<Slide title="Line (2d) Graphs">

- Used to show relation between two variables
- Time-series plot when x-axis is time
- Can be lines or points (scatter plot)
- Categorical comparisons with multiple lines

<Slide title="Box Plot">

- Plots IQR, median, and “range”
- Used to show distributions
- Often by category
- Information dense

</Slide>

<Slide title="Violin Plot">

- Plots distributions like box plot, but smoothed
- Used to show distributions
- Often by category
- Information dense

</Slide>
